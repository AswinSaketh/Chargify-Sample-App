setKinds( \array $kinds ) : \Crucial_Service_Chargify_Transaction

An array of transaction types. Multiple values can be passed in the url, for example: http://example.com?kinds[]=charge&kinds[]=payment&kinds[]=credit

The following is a list of available transaction types.
