currentTab( \string $tab ) : \string

Returns the string "current" if the give tab is the active one. An empty string if not.

I know there is Zend_Navigation but it forces tight coupling of your tabs to your modules/controllers/actions. I don't think navigation tabs should enforce the structure of your application.
