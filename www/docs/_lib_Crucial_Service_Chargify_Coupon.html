find( \int $productFamilyId ) : \Crucial_Service_Chargify_Coupon

You can search for a coupon via the API with the find method. By passing a code parameter, the find record will attempt to locate a coupon that matches that code. This method is useful for validating coupon codes that are entered by a customer. If no coupon is found, a 404 is returned.
