setAmountInCents( \int $amountInCents ) : \Crucial_Service_Chargify_Adjustment

(either 'amount' or 'amount_in_cents' is required) If you use this parameter, you should pass the amount represented as a number of cents, either as a string or integer. For example, $10.00 would be represented as 1000 and -$10.00 would be represented as -1000. If you pass a value for both 'amount' and 'amount_in_cents', the value in 'amount_in_cents' will be used and 'amount' will be discarded.
